uyabuild 0.4.0-phase3

Usage:
  uyabuild [global-options] <command> [command-options]

Bootstrap:
  make bootstrap        build the pure Uya bootstrap binary into bin/uyabuild

Commands:
  build                 analyze, snapshot, and execute supported local actions
  query                 inspect the typed target graph
  plan                  emit the typed IR plus Phase 2 action graph
  version               print the bootstrap version
  help                  show command help

Global options:
  --workspace <path>    override workspace root discovery
  --state-dir <path>    override state directory (default: <workspace>/.uya-build)
  --json                emit diagnostics as JSON
  -h, --help            show help
  --version             print version

Examples:
  uyabuild build //bootstrap:uyabuild
  uyabuild query //...
  uyabuild plan //app:hello --json
